# LUX WATCHES - Premium Luxury Watch Website

## Overview

A cutting-edge, immersive luxury watch website featuring photorealistic 3D watches, interactive configurators, and premium micro-interactions. Built with Three.js, GSAP, and Vite for an unparalleled luxury shopping experience.

## Features

✨ **Interactive 3D Watch Showcase**
- Smooth rotation with mouse tracking
- Dynamic reflections and realistic shadows
- Depth-of-field effects and cinematic lighting
- Realistic PBR materials (brushed steel, polished titanium, rose gold, etc.)

🎨 **Premium 3D Configurator**
- Customize case materials in real-time
- Choose from multiple dial colors
- Select different strap materials
- Instant model updates with realistic reflections

📱 **Responsive Design**
- Dark luxury color palette with gold/silver accents
- Minimalist yet sophisticated layout
- Mobile-friendly interface

⚡ **Advanced Animations**
- GSAP-powered smooth transitions
- Parallax scrolling effects
- Floating particles and light streaks
- Premium micro-interactions

## Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
lux-watches/
├── src/
│   ├── main.js              # Application entry point
│   ├── styles/

│   ├── scenes/

```

## Technologies Used

- **Three.js** - 3D graphics engine
- **GSAP** - Animation framework
- **Vite** - Build tool and dev server
- **SCSS** - Styling
- **WebGL** - GPU-accelerated graphics
